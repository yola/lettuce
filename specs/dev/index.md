Development guidelines
======================

Synopsis
--------

1.  fork and clone the project: see [lettuce repository at
    github.com](https://github.com/gabrielfalcao/lettuce)
2.  :doc:\`/dev/install\`
3.  run the tests: see :doc:\`/dev/testing\`
4.  hack at will
5.  commit, push, etc...
6.  send a pull request
